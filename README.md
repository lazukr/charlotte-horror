# Charlotte Horror

This is a horror adventure 2D pixel game developed using Phaser 3 and Typescript. It is currently a work in progress.

## Ideas
- Ray casting (done)
- Minimal vision (done)
- Flashlight (In progress)
    - will have battery
- Invisible enemies (Not done)
    - interacts with the flashlight as the flashlight will reveal and paralyze them
- Ambient sounds (Not done)
- Levels (Not done)
    - Currently only contains some test levels

## Running
First time

Run `yarn` to let it get all the necessary dependencies.
Then,
```
    yarn start
```
